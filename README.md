# IA & RH — Espace apprenant

Espace apprenant responsive pour le parcours de formation « IA & Ressources humaines ».

## Contenu

- Accès participant par code : `4321`
- Trois journées de formation avec ressources à déposer
- Emplacement YouTube configurable depuis l’interface
- Quiz interactifs et propositions d’applications pédagogiques par journée

## Développement

```bash
npm install
npm run dev
```

## Publication Netlify

Le projet est configuré avec `netlify.toml`. La commande de publication est `npm run build` et le dossier publié est `dist`.
